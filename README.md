# FullStackOpen - Universidad de Helsinki

## Sobre el Curso

Este repositorio contiene soluciones a los ejercicios del curso **FullStackOpen**, una formación intensiva de la Universidad de Helsinki sobre desarrollo web moderno. El programa se centra en la creación de aplicaciones tipo *Single Page Applications* (SPA) utilizando principalmente el ecosistema de JavaScript.

## Estructura del Repositorio

El contenido está organizado por secciones siguiendo el plan de estudios oficial. Para facilitar la navegación y comprensión de cada módulo:

* **Organización por Partes:** Cada nivel del curso (Part 0, Part 1, Part 2, etc.) se encuentra en su propio directorio.
* **Documentación Detallada:** **Cada parte cuenta con un archivo README individual** que describe los ejercicios realizados, los conceptos clave aprendidos y los desafíos técnicos específicos superados en esa etapa.

## Tecnologías Principales

A lo largo del repositorio se exploran y aplican herramientas como:

* **Frontend:** React, Redux, Hooks, Comunicación con APIs.
* **Backend:** Node.js, Express.
* **Base de Datos:** MongoDB.
* **Otros:** Testing (Jest, Cypress), GraphQL, TypeScript y Docker.
